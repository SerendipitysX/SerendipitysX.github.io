[算法]模板！お邪魔してすみません！

2021-03-12 / 2 min read

お邪魔してすみません！
经过半个月间歇性努力，对算法的用力可以说是相当不均匀了。但是时不我待，必须想办法改进一波。
花3s复盘，发现自己有时候会害怕落于人后所以追求数量，做不出来就会相当焦虑，心情很沮丧。就算是总结了，也很少去看，因为不想再经历一遍，可这样导致的后果是挫败与日俱增，而能力却fluctuate around the ground.
所以，自救行动第一波！
在害怕回顾和提高自信中间找到一个平衡点：
形成做题+模板+应用的良性循环吧!至少现在看来是benign的，很多时候本来benign的东西因为自己差劲的行动力都会变得malignant。看到这儿希望未来的肖四四不要笑，因为你最近背的单词太少，而月末要考托福，所以才会阴阳怪气的插英文。

二分查找模板

version 1: 🌵

 while (l < r)
    {
        int mid = (l + r) // 2;
        if (check(mid)):
             r = mid;
        else:
             l = mid + 1;
    }

**version 2:**🌵

while (l < r)
    {
        int mid = l + r + 1 >> 1;
        if (check(mid)) l = mid;
        else r = mid - 1;
    }
