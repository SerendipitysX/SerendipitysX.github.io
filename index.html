冒泡排序

题设：有n个无序数字，需要按一定顺序排列。
时间复杂度：O(n)O(n)O(n)

Key:

  1. 相邻两个数字比大小，无论结果如何，指针都要向后移
  2. 每一趟都会往有序区（自然在末端啦）塞一个数字
  3. 共n−1n-1n−1趟
def Bubblesort(li):
    for i in range(len(li)-1):
        for j in range(len(li)-i-1):
            if li[j] > li[j+1]:
                li[j],li[j+1] = li[j+1],li[j]
    return li
